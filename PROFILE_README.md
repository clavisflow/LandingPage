## About

システム開発・保守に25年以上携わっています。

C#、ASP.NET、Blazorを中心に、VB6、VBA、Classic ASPなど、新旧さまざまな技術に対応しています。長く使われてきた業務システムの現状を整理し、無理に作り替える前に「直す・残す・置き換える」の選択肢を見える形にすることが得意です。

現在も現役でコードを書きながら、診断・改善提案から保守・開発まで支援しています。

## Focus

- レガシーシステムの調査、可視化、改善支援
- Windows業務アプリケーションの保守・モダナイズ
- Excel / Access資産の解析とドキュメント化
- データベース連携、データ同期
- AIを活用した開発プロセスの改善

## Products & Tools

| プロダクト | 概要 |
| --- | --- |
| [SyncCoordinator](https://synco.clavisflow.net/) | SQL Server・MySQL・PostgreSQL間のデータ同期を構築・運用するセルフホスト型プロダクト |
| [Excel Doctor](https://exceldoctor.clavisflow.net/) | Excelファイルの構造、数式、マクロの問題点をブラウザ上でチェック |
| [Excel Compare](https://excelcompare.clavisflow.net/) | Excelファイルの数式、書式、マクロまで比較できる差分確認ツール |
| [Access Explorer](https://accessexplorer.clavisflow.net/) | Accessファイルの構造を可視化し、仕様書として使えるレポートを出力 |
